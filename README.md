# macOS Native Apps List
The following is a curated collection of apps that effectively implement macOS native design.

#### Abbreviations
- **`BS`** Apps that use macOS 11 Big Sur through macOS 15 Sequoia's design
- **`LG`** Apps that use macOS 26 Tahoe and onward's Liquid Glass design
- **`BSM`** Apps that follow a modified version of macOS 11 Big Sur through macOS 15 Sequoia's design
- **`LGM`** Apps that follow a modified version macOS 26 Tahoe and onward's Liquid Glass design
